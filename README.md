# wage-calculation
An exercise in Angular, Node, Bootstrap etc., but mostly Angular.
Parses CSV files (that are in the correct format) on the client side and displays tables with monthly salary info from the parsed data. Hourly wages, coefficients etc. are hardcoded for this exercise.

### Installation
* `git clone https://github.com/rionpy/wage-calculation.git && cd wage-calculation`
* `npm install`
* `npm start`

The app will be running at `localhost:5000`, unless something went awry.
